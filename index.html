🔬 Advanced MapReduce Pipeline

Distributed Data Processing:

Phase 1: MAP
Input: covid_data.csv (90 records)
→ Parse CSV: (country, date, confirmed, deaths, recovered)
→ Emit: <country_key, health_metrics>

Phase 2: SHUFFLE & SORT
→ Group by country: country_key → [values...]
→ Partition data across reducers

Phase 3: REDUCE
→ Aggregate: sum(confirmed), sum(deaths), sum(recovered)
→ Calculate: active_cases = confirmed - deaths - recovered
→ Output: processed_data.json
  • ✅ Scalability: Processes 90 records → 30 country aggregations
  • ✅ Fault Tolerance: Error handling for malformed CSV data
  • ✅ Performance: O(n log n) complexity with efficient grouping
